# Colors

This directory contains reusable color-reference assets for the standards repository.

## ANSI-scene terminal palette

`ansi-scene-terminal-palette.html` and `ansi-scene-terminal-palette.css` define a visual tear sheet for terminal-oriented color usage.

The palette connects:

- display names
- shell tokens
- HEX values
- CSS variables
- intended shell-output semantics

Private research/source screenshots are retained separately and are not required for using the public tear sheet.
