# Shell Prompts

This document defines shell prompt standards at a high level.

Detailed shell prompt standards live under:

```text
shell/prompts/
```

## Prompt styles

```text
operator
system_default
```

## Operator prompt

The operator prompt is the custom prompt style for personal systems and controlled environments.

It should be readable, calm, and operational.

## System default prompt

The system default prompt preserves the operating system or distribution default.

Use this when the default prompt is meaningful, expected, or safer to preserve.

## Rule of thumb

A prompt should orient the operator without becoming the work.
